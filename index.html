Danny Greg

github

ReactiveCocoa

Native apps spend a lot of time waiting and then reacting. We wait for the user to do something in the UI. Wait for a network call to respond. Wait for an asynchronous operation to complete. Wait for some dependent value to change. And then they react.

But all those things—all that waiting and reacting—is usually handled in many disparate ways. That makes it hard for us to reason about them, chain them, or compose them in any uniform, high-level way. We can do better.

That's why we've open-sourced a piece of the magic behind GitHub for Mac: ReactiveCocoa (RAC).

Biography

Danny has been shipping Mac and iOS applications for over 5 years, nearly all of which have been reliant on a web service somewhere down the line. He currently works for GitHub where he builds the Mac app along with many other goodies.

Gordon Murrison

Open Planet

Core Audio

Core Audio is the most comprehensive and powerful audio programming framework ever created and it's built into OS X and iOS. In this talk we'll explore this low-level C-based world and take a look at the technologies that power apps like GarageBand and Logic. We'll also delve into some of the APIs in more detail to see how, where and when they can be used.

Biography

Gordon is co-founder of Open Planet Software. They make Smoovie (a stop motion animation app for Mac and iPad) and work on lots of interesting project for clients including porting popular games such as LEGO Harry Potter and Bioshock2 to the Mac.

Richard Buckle

Sailmaker Systems

Code Coverage on iOS

Code coverage reports can be invaluable for identifying untested code. However, like all metrics, percentage coverage makes a good servant but a bad master. Moreover getting a coverage report from an iOS simulator or device can require some surprising and poorly documented workarounds.

We will identify good and bad ways to use code coverage reports and provide the full recipe for getting them working using Xcode, iOS and Jenkins.

Biography

Richard is a freelance iOS and Mac consultant and the founder of Sailmaker Systems Ltd. He has been programming on Apple hardware since the days of Macintosh System 6.
